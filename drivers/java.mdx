title: "Java driver (JDBC)"
description: "Connect to Apache AGE from Java using the PostgreSQL JDBC driver and the AGE agtype extension."
---

The Java driver adds agtype parsing support on top of the standard [PostgreSQL JDBC driver](https://jdbc.postgresql.org/). It registers the `Agtype` class as a custom type so that Cypher query results can be retrieved directly as `Agtype` objects, giving you structured access to vertex labels, edge types, and property maps.

## Prerequisites

The following JAR files must be on your classpath:

- [Gradle](https://gradle.org/install/) build tool
- [PostgreSQL JDBC driver](https://jdbc.postgresql.org/download/)
- [antlr4-4.9.2-complete](https://repo1.maven.org/maven2/org/antlr/antlr4/4.9.2/)
- [commons-lang3](http://www.java2s.com/Code/Jar/c/Downloadcommonlang3jar.htm)
- [commons-text-1.6](http://www.java2s.com/ref/jar/download-commonstext16jar-file.html)

<Note>
Unzip any JAR files that arrive in a compressed archive before adding them to your classpath.
</Note>

## Building the driver

Clone the repository and run the Gradle assemble task:

```bash
git clone https://github.com/apache/age.git
cd age/drivers/jdbc

gradle assemble
```

The build produces a JAR at `age/drivers/jdbc/lib/build/libs/lib.jar`. Add this file to your project's classpath.

## Connecting and configuring AGE

Open a connection using the standard JDBC `DriverManager`, unwrap it to `PgConnection`, and register the `Agtype` custom type. Then run the three AGE setup statements:

```java
import org.apache.age.jdbc.base.Agtype;
import org.postgresql.jdbc.PgConnection;

import java.sql.*;

public static PgConnection makeConnection() throws SQLException {
    String DB_URL = "jdbc:postgresql://localhost:5432/demo";
    String USER   = "postgres";
    String PASS   = "pass";

    PgConnection connection = DriverManager
        .getConnection(DB_URL, USER, PASS)
        .unwrap(PgConnection.class);

    // Register the agtype custom type
    connection.addDataType("agtype", Agtype.class);

    // Configure AGE for this session
    Statement statement = connection.createStatement();
    statement.execute("CREATE EXTENSION IF NOT EXISTS age;");
    statement.execute("LOAD 'age'");
    statement.execute("SET search_path = ag_catalog, \"$user\", public;");

    return connection;
}
```

## Creating a graph and nodes

Use standard JDBC `Statement.execute()` to issue Cypher queries wrapped in the `cypher()` SQL function:

```java
public static void createGraph(PgConnection connection) throws SQLException {
    Statement statement = connection.createStatement();

    // Create the graph
    statement.execute("SELECT create_graph('demo_graph');");

    // Add Person nodes
    statement.execute("SELECT * FROM cypher('demo_graph', $$ CREATE (n:Person {name : 'imran', bornIn : 'Pakistan'}) $$) AS (a agtype);");
    statement.execute("SELECT * FROM cypher('demo_graph', $$ CREATE (n:Person {name : 'ali',   bornIn : 'Pakistan'}) $$) AS (a agtype);");
    statement.execute("SELECT * FROM cypher('demo_graph', $$ CREATE (n:Person {name : 'james', bornIn : 'US'})      $$) AS (a agtype);");

    // Add Country nodes
    statement.execute("SELECT * FROM cypher('demo_graph', $$ CREATE (n:Country{name : 'Pakistan'}) $$) AS (a agtype);");
    statement.execute("SELECT * FROM cypher('demo_graph', $$ CREATE (n:Country{name : 'US'})       $$) AS (a agtype);");

    // Create BORNIN relationships
    statement.execute(
        "SELECT * FROM cypher('demo_graph', $$ " +
        "MATCH (a:Person), (b:Country) WHERE a.bornIn = b.name " +
        "CREATE (a)-[r:BORNIN]->(b) RETURN r " +
        "$$) as (r agtype);"
    );
}
```

## Querying and reading Agtype results

Execute a `SELECT` query and retrieve each row as an `Agtype` object using `ResultSet.getObject()`:

### Raw Agtype value

```java
public static void getResult(PgConnection connection) throws SQLException {
    Statement stmt = connection.createStatement();
    ResultSet rs = stmt.executeQuery(
        "SELECT * from cypher('demo_graph', $$ MATCH (n) RETURN n $$) as (n agtype);"
    );

    while (rs.next()) {
        Agtype returnedAgtype = rs.getObject(1, Agtype.class);
        System.out.println(returnedAgtype.getValue());
    }
}
```

### Extracting label and properties

Use `getMap()` on the `Agtype` object to access named fields:

```java
public static void getObjectsFromResult(PgConnection connection) throws SQLException {
    Statement stmt = connection.createStatement();
    ResultSet rs = stmt.executeQuery(
        "SELECT * from cypher('demo_graph', $$ MATCH (n) RETURN n $$) as (n agtype);"
    );

    while (rs.next()) {
        Agtype returnedAgtype = rs.getObject(1, Agtype.class);
        String nodeLabel = returnedAgtype.getMap().getObject("label").toString();
        String nodeProp  = returnedAgtype.getMap().getObject("properties").toString();
        System.out.println("Vertex : " + nodeLabel + ", \tProps : " + nodeProp);
    }
}
```

Example output:

```text
Vertex : Person,  Props : {bornIn=Pakistan, name=imran}
Vertex : Person,  Props : {bornIn=Pakistan, name=ali}
Vertex : Person,  Props : {bornIn=US, name=james}
Vertex : Country, Props : {name=Pakistan}
Vertex : Country, Props : {name=US}
```

## Complete example

The sample file `AgeBasic.java` demonstrates a full lifecycle — connect, create graph, query, drop graph, and close:

```java
import org.apache.age.jdbc.base.Agtype;
import org.postgresql.jdbc.PgConnection;

import java.sql.*;

public class AgeBasic {

    public static PgConnection connection;
    static final String DB_URL = "jdbc:postgresql://localhost:5432/demo";
    static final String USER   = "postgres";
    static final String PASS   = "pass";

    public static void main(String[] args) {
        try {
            makeConnection();

            if (graphExist("demo_graph"))
                dropGraph();

            createGraph();
            getResult();
            getObjectsFromResult();

            dropGraph();
            closeConnection();
        } catch (Exception e) {
            System.out.println(e.getMessage());
        }
    }

    public static boolean graphExist(String graph_name) throws SQLException {
        PreparedStatement ps = connection.prepareStatement(
            "SELECT * FROM ag_catalog.ag_graph WHERE name = ?"
        );
        ps.setString(1, graph_name);
        ps.executeQuery();
        ResultSet rs = ps.getResultSet();
        return !(!rs.isBeforeFirst() && rs.getRow() == 0);
    }

    public static void dropGraph() throws SQLException {
        Statement statement = connection.createStatement();
        statement.execute("SELECT * FROM ag_catalog.drop_graph('demo_graph', true);");
    }

    public static void closeConnection() throws SQLException {
        connection.close();
    }
}
```

## Agtype object model

| Method | Returns | Description |
|--------|---------|-------------|
| `agtype.getValue()` | `Object` | The raw parsed agtype value |
| `agtype.getMap()` | `AgtypeMap` | Vertex or edge as a key/value map |
| `agtypeMap.getObject("label")` | `Object` | The label or relationship type |
| `agtypeMap.getObject("properties")` | `Object` | Property map |
