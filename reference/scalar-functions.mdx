---
title: "Scalar functions"
description: "Cypher scalar functions implemented in ag_catalog for inspecting vertices, edges, paths, and values."
---

Scalar functions return a single `agtype` value per invocation. They are called inside Cypher queries and are mapped to the corresponding `ag_catalog` SQL functions listed below.

All SQL functions are in the `ag_catalog` schema. Functions marked **IMMUTABLE** can be used in indexes; functions marked **STABLE** are safe inside transactions.

---

## Graph element functions

### id

```sql
ag_catalog.age_id(agtype) → agtype
```

Returns the internal ID of a vertex or edge as an `agtype` integer.

| Cypher name | SQL name        | Volatility |
| ----------- | --------------- | ---------- |
| `id()`      | `age_id(agtype)`| IMMUTABLE  |

```sql
SELECT * FROM ag_catalog.cypher('my_graph', $$
    MATCH (n:Person)
    RETURN id(n)
$$) AS (node_id agtype);
```

## Predicate functions

### exists

```sql
ag_catalog.age_exists(agtype) → boolean
```

Returns `true` if the given property or pattern exists, `false` otherwise. Returns a PostgreSQL `boolean`, not `agtype`.

| Cypher name | SQL name              | Volatility |
| ----------- | --------------------- | ---------- |
| `exists()`  | `age_exists(agtype)`  | STABLE     |

```sql
SELECT * FROM ag_catalog.cypher('my_graph', $$
    MATCH (n:Person)
    WHERE exists(n.email)
    RETURN n.name
$$) AS (name agtype);
```

---

### isEmpty

```sql
ag_catalog.age_isempty(agtype) → boolean
```

Returns `true` if a list or map is empty, `false` otherwise. Returns a PostgreSQL `boolean`.

| Cypher name  | SQL name                | Volatility |
| ------------ | ----------------------- | ---------- |
| `isEmpty()`  | `age_isempty(agtype)`   | IMMUTABLE  |

```sql
SELECT * FROM ag_catalog.cypher('my_graph', $$
    MATCH (n:Person)
    WHERE isEmpty(n.tags)
    RETURN n.name
$$) AS (name agtype);
```
