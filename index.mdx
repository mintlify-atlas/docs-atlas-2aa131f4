## What is Apache AGE?

Apache AGE is a PostgreSQL extension that enables you to use a graph database on top of your existing relational database. The name stands for **A Graph Extension**, inspired by Bitnine's AgensGraph.

The core idea: a **single storage** that handles both relational and graph data. You can use standard ANSI SQL alongside openCypher — one of the most popular graph query languages — in the same database and even the same query.

## Key features

- **openCypher support** — write expressive graph queries with `MATCH`, `CREATE`, `MERGE`, `DELETE`, and more
- **Hybrid querying** — call `cypher()` inside any SQL query; join graph results with relational tables
- **Multiple graphs** — create and query multiple named graphs in one database
- **Property indexes** — B-tree and GIN indexes on vertex and edge properties for fast lookups
- **Full PostgreSQL compatibility** — backups, replication, roles, and extensions all work as expected
- **Language drivers** — official drivers for Python, Go, Java (JDBC), and Node.js

## Get started

<Steps>
  <Step title="Install AGE">
    Pull the Docker image or build from source. See the [installation guide](/installation) for all options.

    ```bash
    docker pull apache/age
    docker run --name age -p 5455:5432 \
      -e POSTGRES_USER=postgresUser \
      -e POSTGRES_PASSWORD=postgresPW \
      -e POSTGRES_DB=postgresDB \
      -d apache/age
    ```
  </Step>
  <Step title="Load the extension">
    Connect to your database and enable AGE for your session.

    ```sql
    CREATE EXTENSION age;
    LOAD 'age';
    SET search_path = ag_catalog, "$user", public;
    ```
  </Step>
  <Step title="Create a graph">
    Use `create_graph` to create a named graph, then run your first Cypher query.

    ```sql
    SELECT create_graph('my_graph');

    SELECT * FROM cypher('my_graph', $$
      CREATE (n:Person {name: 'Alice', age: 30})
      RETURN n
    $$) AS (n agtype);
    ```
  </Step>
  <Step title="Query your data">
    Match patterns, traverse relationships, and return results.

    ```sql
    SELECT * FROM cypher('my_graph', $$
      MATCH (a:Person)-[r:KNOWS]->(b:Person)
      RETURN a.name, b.name, r.since
    $$) AS (person1 agtype, person2 agtype, since agtype);
    ```
  </Step>
</Steps>

<Note>
  Apache AGE supports PostgreSQL versions 11 through 18. Make sure your PostgreSQL version is compatible before installing.
</Note>
